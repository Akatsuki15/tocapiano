Descarga las dependencias del proyecto
```
npm install
```

Monta en modo desarrollo mi app para probarla
```
npm run dev
```

Construir la aplicación
```
npm run build
```

Instalar una dependencia en particular
```
npm intall xxx
```

Deploy Github
```
npm run deploy
```
